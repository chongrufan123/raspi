# raspi
这个仓库记录我对于树莓派做的一些奇奇怪怪的事情, 将以类似博客的形式记载, 记载在鼓捣树莓派的过程中遇到的困难和解决方法等

## 树莓派配置
型号:raspi 4代B型

CPU:Quad core Cortex-A72 (ARM v8) 64-bit @ 1.5GHz

内存:2GB

## 其他情况说明
本人因为没有路由器的关系, 一般是将树莓派通过网线和笔记本电脑相连, 使得他们在同一个局域网下, 然后通过ssh连接树莓派,
对树莓派进行一些操作  
配件拥有一个SSD1306为驱动芯片的4接口128*64的OLED, 可以通过i2c和树莓派相连, 显示一些简单的信息
